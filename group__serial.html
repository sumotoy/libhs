Enumerator
HS_SERIAL_CSIZE_7BITS 

Use 7 bits in the bytes transmitted and received, instead of 8 bytes.

HS_SERIAL_CSIZE_6BITS 

Use 6 bits in the bytes transmitted and received, instead of 8 bytes.

HS_SERIAL_CSIZE_5BITS 

Use 5 bits in the bytes transmitted and received, instead of 8 bytes.

HS_SERIAL_PARITY_ODD 

Use an odd scheme for parity, instead of no parity.

HS_SERIAL_PARITY_EVEN 

Use an even scheme for parity, instead of no parity.

HS_SERIAL_STOP_2BITS 

Use two stop bits instead of one.

HS_SERIAL_FLOW_XONXOFF 

Enable XON/XOFF (software) flow control on input.

HS_SERIAL_FLOW_RTSCTS 

Enable RTS/CTS (hardware) flow control.

HS_SERIAL_CLOSE_NOHUP 

Keep the DTR line high on close (not supported on Windows).

Set the parameters associated with a serial device.

The change is carried out immediately, before the buffers are emptied.

Parameters
hOpen serial device handle.
rateSerial baud rate, see hs_serial_rate.
flagsSerial connection settings, see hs_serial_flag.
Returns
This function returns 0 on success, or a negative hs_err code.

hs_serial_rate for acceptable baud rates.

uint8_t *  buf,
size_t  size,
int  timeout 
)

Read bytes from a serial device.

Read up to size bytes from the serial device. If no data is available, the function waits for up to timeout milliseconds. Use a negative value to wait indefinitely.

Parameters
hDevice handle.
[out]bufData buffer.
sizeSize of the buffer.
timeoutTimeout in milliseconds, or -1 to block indefinitely.
Returns
This function returns the number of bytes read, or a negative hs_err code on error.

Send bytes to a serial device.

Write up to size bytes to the device. This is a blocking function, but it may not write all the data passed in.

Parameters

bufData buffer.
sizeSize of the buffer.
Returns
This function returns the number of bytes written, or a negative hs_err code on error.
